# Portfólio - Jairo Junior

Portfólio profissional de Jairo Junior, Software Engineer especializado em Backend Development com Kotlin, Micronaut e Clean Architecture.

## 🚀 Tecnologias Utilizadas

- **React 18** - Biblioteca JavaScript para interfaces
- **TypeScript** - Superset do JavaScript com tipagem estática
- **Styled Components** - CSS-in-JS para estilização
- **Framer Motion** - Biblioteca de animações
- **React Icons** - Ícones para React

## 📋 Pré-requisitos

- Node.js (versão 16 ou superior)
- npm ou yarn

## 🔧 Instalação

1. Clone o repositório:
```bash
git clone <url-do-repositorio>
cd portfolio
```

2. Instale as dependências:
```bash
npm install
```

3. Execute o projeto em modo de desenvolvimento:
```bash
npm start
```

O projeto será aberto automaticamente no navegador em `http://localhost:3000`.

## 📦 Scripts Disponíveis

- `npm start` - Executa o projeto em modo de desenvolvimento
- `npm build` - Cria a versão de produção
- `npm test` - Executa os testes
- `npm eject` - Ejecta a configuração do Create React App
